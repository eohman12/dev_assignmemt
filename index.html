ABOUT THIS PAGE

My name is Emily Ohman, a rising junior at UC Berkeley. This website was created for my multimedia journalism class at UC Berkeley, and other than looking cool, it doesn't really have any other purpose.

"Maybe someday I'll be able to use this page as an actual website to showcase my journalistic work and art!" —Me

I work for the Daily Californian, which has a notoriously terrible website. Seeing how much fun it was to work with HTML and CSS, even though I'm a notoriously terrible coder, has really made me feel like I could give it a makeover.

WHAT I WOULD USE THIS WEBSITE FOR

Before I redo the Daily Cal's website, though, it would be cool to get this one working just for myself! I've learned so much about multimedia and visual storytelling from this minor, especially about the editing processes behind them. I'd love to have a place to publish some of the projects I work on.

"I see this as a practical thing for your future." —Lisi Ludwig, friend and classmate

I never knew how much of a creative outlet website building could be, and though I have quite a lot to learn, I think this could be a great way to sharpen my visual art skills.
